# 해외 뉴스 요약

1. 매일 일정 시간마다 당일의 뉴스를 크롤링한다.
2. runpod의 serverless로 배포된 Gemma2 9B 모델이 뉴스를 요약한다.
3. 요약된 결과를 Discord bot 채널에서 새로운 스레드를 생성한 뒤 전달한다.
